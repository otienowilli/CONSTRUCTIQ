## Getting Started

### Creating an Account

1. Navigate to the CONSTRUCTIQ web dashboard
2. Click "Sign Up" or "Register"
3. Fill in your details:
   - Full Name
   - Email Address
   - Password
   - Role (Engineer, Supervisor, Worker)
4. Click "Create Account"
5. Verify your email address

### Logging In

1. Go to the login page
2. Enter your email and password
3. Click "Login"
4. You'll be redirected to the dashboard

## Project Management

### Creating a New Project

1. Click "New Project" button
2. Fill in project details:
   - Project Name
   - Description
   - Type (Building, Road, Bridge, etc.)
   - Location (GPS coordinates)
   - Budget
   - Start Date
   - End Date
3. Click "Create Project"

### Viewing Projects

1. Navigate to "Projects" section
2. View list of all your projects
3. Click on a project to see details:
   - Project information
   - Material calculations
   - Site analysis
   - Progress reports
   - Team members

### Updating Project Status

1. Open project details
2. Click "Update Status"
3. Select new status:
   - Planning
   - In Progress
   - On Hold
   - Completed
4. Add notes if needed
5. Click "Save"

## Material Calculator

The material calculator helps you estimate material requirements for various construction tasks.

### Concrete Calculator

1. Go to "Material Calculator"
2. Select "Concrete"
3. Enter dimensions:
   - Length (meters)
   - Width (meters)
   - Height/Thickness (meters)
4. Select concrete grade (C15, C20, C25, C30)
5. Click "Calculate"
6. View results:
   - Cement (bags)
   - Sand (m³)
   - Ballast/Aggregates (m³)
   - Water (liters)
   - Total cost

### Brickwork Calculator

1. Select "Brickwork"
2. Enter:
   - Wall length (meters)
   - Wall height (meters)
   - Wall thickness (9-inch, 4.5-inch)
3. Select mortar ratio (1:4, 1:6)
4. Click "Calculate"
5. View:
   - Number of bricks
   - Cement (bags)
   - Sand (m³)
   - Total cost

### Other Calculators

- **Plastering** - Calculate plaster materials
- **Painting** - Estimate paint requirements
- **Excavation** - Calculate excavation volume and cost
- **Road Construction** - Estimate road materials
- **Roofing** - Calculate roofing materials

### Saving Calculations

1. After calculating, click "Save to Project"
2. Select project
3. Add notes
4. Click "Save"

## Site Analysis

### Uploading Site Images

1. Go to "Site Analysis"
2. Select project
3. Click "Upload Images"
4. Choose images from your device
5. Add location (GPS coordinates)
6. Click "Analyze"

### AI Soil Analysis

1. Upload soil image
2. AI will analyze:
   - Soil type (Clay, Sand, Loam, etc.)
   - Moisture level
   - Composition
   - Bearing capacity
3. View recommendations:
   - Foundation type
   - Soil treatment
   - Construction precautions

### Viewing Analysis Results

- Soil type and properties
- Construction recommendations
- Foundation suggestions
- Risk assessment

## Reports

### Generating Reports

1. Go to "Reports"
2. Click "Generate Report"
3. Select:
   - Project
   - Report type (Progress, Cost, Materials)
   - Period (Daily, Weekly, Monthly)
4. Click "Generate"

### Report Types

- **Progress Report** - Project progress and milestones
- **Cost Report** - Budget and expenses
- **Materials Report** - Material usage and inventory
- **Safety Report** - Safety incidents and compliance

### Exporting Reports

1. Open report
2. Click "Export"
3. Choose format (PDF, Excel)
4. Download file

## Tips and Best Practices

### Material Calculations

- Always add 5-10% wastage
- Verify local material prices
- Save calculations for future reference
- Review AI recommendations

### Site Analysis

- Take clear, well-lit photos
- Capture multiple angles
- Include scale reference
- Update regularly

### Project Management

- Update status regularly
- Document all changes
- Keep team informed
- Review reports weekly

## Troubleshooting

### Can't Login

- Check email and password
- Reset password if needed
- Clear browser cache
- Contact support

### Calculator Not Working

- Check internet connection
- Verify all fields are filled
- Refresh page
- Try different browser

### Images Not Uploading

- Check file size (max 10MB)
- Use supported formats (JPG, PNG)
- Check internet connection
- Compress large images
