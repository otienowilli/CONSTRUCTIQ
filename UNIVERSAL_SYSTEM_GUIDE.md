## ➕ **ADD CUSTOM MATERIALS**

### **How It Works:**
1. Tap "Add Material" button
2. Enter material name
3. Select category or create new
4. Choose unit of measurement
5. Enter price per unit
6. Optionally add density & description
7. Save - material is now available!

### **Supported Units:**
- Weight: kg, ton, g, lb
- Volume: m³, L, ft³, gal
- Area: m², ft², hectare, acre
- Length: m, cm, mm, ft, in
- Pieces: pcs, units

---

## 🤖 **ADD CUSTOM MACHINES**

### **How It Works:**
1. Tap "Add Machine" button
2. Enter machine details (name, model, serial)
3. Select category or create custom
4. Choose power type
5. **Add sensors using codes**
6. Save - machine is ready!

### **Sensor Installation:**
- Select from 40+ standard sensor codes
- Add custom sensor codes (format: `XXX-###`)
- Multiple sensors per machine supported
- Sensors auto-detect when connected

## 🎛️ **UNIVERSAL MACHINE CONTROL**

### **Command System (11 Commands)**
- `CMD-001` - START
- `CMD-002` - STOP (with confirmation)
- `CMD-003` - PAUSE
- `CMD-004` - RESUME
- `CMD-005` - EMERGENCY STOP
- `CMD-006` - RESET
- `CMD-007` - CALIBRATE
- `CMD-008/009` - SPEED UP/DOWN
- `CMD-010/011` - AUTO/MANUAL MODE

### **Works With ANY Machine:**
As long as the machine has:
1. IoT connectivity (WiFi/4G/5G)
2. MQTT support
3. Sensors installed with proper codes
4. Compatible control unit
